# wkinga Beats Backend

Backend for wangchuk kinga beats made with Nest started framework.
## Installation

```bash
$ npm install
```

## Database Setup
1. install postgres
1. install pgAdmin 4
1. create a server called localhost with username: postgres, password: postgres
1. then create a database called wkinga
## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
